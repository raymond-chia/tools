# 戰鬥統計功能

在模擬模式中加入戰鬥統計，協助設計角色技能。

---

## 傷害統計

| 項目 | 設計用途 |
|------|----------|
| 造成傷害 | 評估單位/技能的輸出能力 |
| 受到傷害 | 評估單位的生存壓力，調整 HP/防禦 |
| 最高單次傷害 | 判斷是否有過強的爆發技能 |
| 治療量 | 平衡治療技能與傷害技能的效益 |

## 命中統計

| 項目 | 設計用途 |
|------|----------|
| 命中率 | 若低於 50%，需調整 accuracy 或敵人 evasion |
| 爆擊次數 | 確認爆擊率符合預期 |
| 閃避次數 | 評估閃避系角色的生存能力 |
| 格擋次數 | 評估格擋系角色的減傷效益 |
| 完全命中/閃避 | 評估 5%/95% 門檻產生的極端結果頻率 |
| 豁免成功/失敗 | 平衡控制技能的效果 |

## 資源統計

| 項目 | 設計用途 |
|------|----------|
| MP 消耗 | 計算每點 MP 的傷害效率，平衡技能 cost |
| 行動次數 | 了解戰鬥長度，調整技能冷卻/消耗 |
| 反應次數 | 評估反應技能的觸發頻率是否合理 |
| 移動格數 | 評估移動力對戰鬥的影響 |

## 技能統計（按單位分組）

| 項目 | 設計用途 |
|------|----------|
| 使用次數 | 找出偏好/冷門技能，判斷是否需要調整 |
| 技能命中率 | 單一技能命中率過低時需調整 |
| 技能平均傷害 | 比較同類技能的效益是否平衡 |
| 技能爆擊次數 | 確認特定技能的爆擊表現 |
