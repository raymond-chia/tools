# 戰鬥統計功能

在模擬模式中加入戰鬥統計，協助設計角色技能。

---

## 總回合數

戰鬥總回合數，用於評估戰鬥長度。

## 傷害統計

| 項目           | 設計用途                         |
| -------------- | -------------------------------- |
| 造成傷害       | 評估單位/技能的輸出能力          |
| 受到傷害       | 評估單位的生存壓力，調整 HP/防禦 |
| 最高單次傷害   | 判斷是否有過強的爆發技能         |
| 治療量（造成） | 平衡治療技能與傷害技能的效益     |
| 治療量（受到） | 評估單位的治療依賴               |

## 命中統計

| 項目         | 設計用途                  |
| ------------ | ------------------------- |
| 攻擊次數     | 總攻擊嘗試次數            |
| 命中次數     | 成功命中次數              |
| 爆擊次數     | 確認爆擊率符合預期        |
| 完全命中次數 | 評估 95% 以上命中率的效果 |
| 被攻擊次數   | 單位受到攻擊的頻率        |
| 閃避次數     | 評估閃避系角色的生存能力  |
| 格擋次數     | 評估格擋系角色的減傷效益  |
| 完全閃避次數 | 評估 5% 以下閃避率的效果  |
| 豁免成功次數 | 平衡控制技能的效果        |
| 豁免失敗次數 | 平衡控制技能的效果        |

## 資源統計

| 項目     | 設計用途                              |
| -------- | ------------------------------------- |
| MP 消耗  | 計算每點 MP 的傷害效率，平衡技能 cost |
| 行動次數 | 了解戰鬥長度，調整技能冷卻/消耗       |
| 反應次數 | 評估反應技能的觸發頻率是否合理        |
| 移動格數 | 評估移動力對戰鬥的影響                |

## 技能統計（按單位分組）

| 項目     | 設計用途                            |
| -------- | ----------------------------------- |
| 使用次數 | 找出偏好/冷門技能，判斷是否需要調整 |
| 總傷害   | 技能造成的總傷害量                  |
| 命中次數 | 技能命中次數                        |
| 攻擊次數 | 技能攻擊嘗試次數                    |
| 爆擊次數 | 確認特定技能的爆擊表現              |

## 死亡統計

| 項目         | 設計用途                               |
| ------------ | -------------------------------------- |
| 擊殺數       | 評估單位的殺傷能力                     |
| 死亡數       | 確認單位是否死亡（0 或 1）             |
| 各技能擊殺數 | 分析哪些技能更有效於擊殺               |
| 死亡原因     | 了解單位死亡的方式（技能、燃燒、坑洞） |
