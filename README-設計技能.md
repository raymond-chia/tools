# 擴展定位系統（12個核心定位）

## 輸出類（4個）

1. 持續輸出（Sustained DPS）

核心：依賴普攻或基礎傷害技能，造成穩定持續傷害  
技能設計：攻速提升、攻擊力提升、多段傷害、被動傷害倍率  
典型組合：持續輸出 + 生存  
例：射手、持續法傷單位

2. 爆發輸出（Burst Damage）

核心：短時間內造成極高傷害，通常需要逃脫  
技能設計：高倍率技能、技能組合、一次性傷害倍增  
典型組合：爆發輸出 + 位移（逃脫）  
例：刺客、砲台法師

3. 範圍輸出（Area Damage）

核心：技能造成群體傷害，適合人多的戰場  
技能設計：群體傷害技能、多次傷害、鏈式傷害  
典型組合：範圍輸出 + 控制（啟動範圍）或 + 生存  
例：團戰法師、砲塔坦克

4. 持續傷害（DoT / Damage Over Time）

核心：透過狀態效果造成遞延傷害，堆疊效果  
技能設計：中毒、灼燒、出血、疊層機制  
典型組合：持續傷害 + 軟控（減速配合疊層）  
例：毒系單位、詛咒系施法者  
戰棋優勢：不需要直接接觸，可遠程削弱

## 防守類（3個）

5. 坦克（Tank）

核心：承受傷害，保護隊友，通常有群體控制  
技能設計：減傷、護盾、反傷、群體控制  
典型組合：坦克 + 控制 或 + 進場  
例：護甲高、血量多的前排

6. 規避型（Evasion / Dodge）

核心：透過躲避或格擋來減少傷害，而非直接承受  
技能設計：閃避、無敵幀、反彈傷害、調整位置  
典型組合：規避型 + 反制 或 + 位移  
例：影子刺客、靈活位移者  
與坦克區別：坦克扛傷害，規避型避傷害

7. 生命恢復（Healer / Sustain）

核心：透過回血或護盾維持生存，持久戰能力強  
技能設計：治療、護盾、回血比例、吸血  
典型組合：生命恢復 + 輔助 或 + 持續輸出  
例：聖光牧師、吸血鬼  
戰棋特性：長期戰損耗對手資源

## 控制與支援類（4個）

8. 硬控制（Hard Control）

核心：禁止敵人行動的技能（定身、擊飛、暈眩）  
技能設計：眩暈、定身、擊飛、恐懼、禁用技能  
典型組合：硬控制 + 進場 或 + 輔助  
例：鎖鏈法師、重擊戰士  
價值：確保隊友的輸出時間

9. 軟控制（Soft Control）

核心：限制敵人行動但不完全禁止（減速、沉默、禁招）  
技能設計：減速、移動限制、技能冷卻延長、沉默  
典型組合：軟控制 + 位移 或 + 持續輸出  
例：冰霜法師、陷阱獵人  
戰棋應用：配合位置控制，困住敵人

10. 輔助（Support / Utility）

核心：增強隊友能力，削弱敵人，但自己輸出低  
技能設計：增傷、增速、增防、視野、減敵傷害  
典型組合：輔助 + 任何定位  
例：吟遊詩人、聖職者  
戰棋特性：乘法加成，後期越來越值錢

11. 位置控制（Zone Control / Zoner）

核心：佔據棋盤關鍵位置，限制敵人移動，創造有利陣地  
技能設計：範圍禁入、地形改變、位置標記、減速地帶  
典型組合：位置控制 + 軟控 或 + 範圍輸出  
例：領地術士、陣地幻術師  
戰棋優勢：與棋盤位置機制深度結合
